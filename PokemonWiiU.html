GB, GBC, GBA, ROM Hack Games

To Access these games power on the Wii U, pressing the Wii U menu button by using the Wii U Game Pad (with the screen)

  • Scroll along till you see Haxchi
  • Press Haxchi and Open
  • It will load and self-close, that's fine
  • Now press Homebrew Channel and Open
  • One of the items we can open is MGBA, click on this and press Load
  • Once in press Load Content
  • The path is -> SD:/ROMS/ once you're in there, just open the game you want to play
  • Press the home button when you're done and press B until you're at the initial menu screen. Press Quit Retroarch.

Gamecube Games

To Access these games plug in the Sensor Bar and have the Wii Remote ready.

Power on the Wii U, pressing the Wii U menu button by using the Wii U Game Pad (with the screen)

  • Press the Wii Channel button and then Load
  • Using the Wii Remote, press the Homebrew Channel button and then Load
  • Open Nintendont using the Wii Remote and press Load
  • Use the Wii U Pro Controller here to select the SD card
  • Pick the game you want
  • Press the home button a couple of times when you're done and press Return to System Menu to quit.
